# 🎆 赛博电子烟花

一个可以在浏览器中发射烟花的趣味网页应用。

## 🌐 在线访问

**https://charse0502.github.io/firework/**

## ✨ 功能特性

- 👆 点击/触摸屏幕发射烟花
- 🎆 5种烟花形状：球形、圆环、五角星、爱心、文字
- ⚙️ 可调节粒子数量、重力系数、拖尾长度
- 🎨 多种颜色基调：彩色、红色、蓝色、绿色、金色
- 📝 支持自定义文字（逗号分隔多行）
- ✨ 自动发射烟花

## 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/Charse0502/firework.git

# 直接用浏览器打开 index.html
# 或者用 Python 启动本地服务器
cd firework
python3 -m http.server 8080
```

然后访问 http://localhost:8080

## 📱 支持设备

- 电脑端（鼠标点击）
- 移动端（触摸发射）
- 支持拖尾滑动发射

## 🔧 技术栈

- HTML5 Canvas
- JavaScript 粒子系统
- GitHub Pages 托管
