# 🎆 赛博电子烟花

一个可以在浏览器中发射烟花的趣味网页应用。

## 🌐 在线访问

**https://charse0502.github.io/firework/**

## ✨ 功能特性

- 👆 点击/触摸屏幕发射烟花
- 🤚 手势控制 - 捏合手指发射烟花（需摄像头权限）
- 🎆 6种烟花形状：球形、圆环、五角星、爱心、文字、随机
- ⚙️ 可调节粒子数量、文字大小、重力系数、拖尾长度
- 🎨 多种颜色基调：彩色/随机、单色固定
- 📝 支持自定义文字（多行文案，每次随机播放）
- 🔥 自动发射烟花

## 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/Charse0502/firework.git

# 直接用浏览器打开 index.html
# 或者用 Python 启动本地服务器
cd firework
python3 -m http.server 8080
```

然后访问 http://localhost:8080

## 📱 支持设备

- 电脑端（鼠标点击）
- 移动端（触摸发射）
- 支持手势控制（需摄像头权限）

## 🔧 技术栈

- HTML5 Canvas
- JavaScript 粒子系统
- MediaPipe 手势识别
- GitHub Pages 托管
